Steps to compile and run gcov:
1. To compile, run the command "make." The program will run
on its own after executing this command.
2. Once program is compiled, run the program once again to
with the command "./testme" to generate the necessary files
in order to run gcov in the next step.
3. After the program finishing executing, run the command
"gcov testme.c" to see the coverage.
4. To clean directory of created files, run the command
"make clean" and all created files will be deleted.
